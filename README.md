# Installation

```
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```


## Clone the repo in .config directory.

```
git clone git clone https://github.com/DaneX3089/nvim ~/.config/nvim
```
